# Quick Setup Guide

## Step 1: Install Node.js

**Download and install Node.js LTS from:** https://nodejs.org/

✅ During installation, make sure "Add to PATH" is checked

## Step 2: Restart Your Terminal/IDE

Close and reopen your terminal so PATH changes take effect.

## Step 3: Verify Installation

```bash
node --version
npm --version
```

Both should show version numbers.

## Step 4: Install Dependencies

```bash
cd frontend
npm install
```

This will take 1-2 minutes to download all packages.

## Step 5: Start Development Server

```bash
npm run dev
```

The app will open at `http://localhost:3000`

## That's it! 🎉

If you encounter any issues, see `NODE_INSTALLATION.md` for detailed troubleshooting.
